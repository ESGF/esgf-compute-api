Named Parameter.

Describes a parameter to be passed to an Operation.

A NamedParameter with a single value.

>>> axis = NamedParameter('axis', 'time')

A NamedParameter with multiple values.

>>> axes = NamedParameter('axes', 'latitude', 'longitude')
